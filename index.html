Install App
Add Kez Media to your home screen so it opens like a real app — fullscreen, no browser bar.
✅ Kez Media is already installed on your device!
📲 Install on iPhone / iPad
1️⃣ Open this page in Safari
2️⃣ Tap the Share button at the bottom of the screen
3️⃣ Scroll and tap "Add to Home Screen"
4️⃣ Tap Add — done! 🎉
📲 Install on Android
1️⃣ Open this page in Chrome
2️⃣ Tap the ⋮ menu at the top right
3️⃣ Tap "Add to Home screen"
4️⃣ Tap Add — done! 🎉
